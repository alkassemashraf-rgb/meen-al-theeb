# Mission 13.1 — Prompt Format & Tone System

## Summary

Replaces the question format ("من أكثر شخص...؟") with a declarative accusation format. Prompts now feel like the group is confronting a shared truth rather than answering a quiz. This produces faster recognition, stronger reactions, and more social energy.

Supersedes: Mission 4 §8 (Pre-Commit Review Checklist)

---

## 1. Why Accusation > Question

The original question format asked the group to think. The new format tells the group what they already know.

| Old Format | New Format |
|---|---|
| Asks who fits a trait | Declares that someone fits a trait |
| Players deliberate | Players immediately recognize |
| Ends with `؟` | Ends with no punctuation or `…` |
| Neutral, observational | Direct, confrontational, warm |

A good prompt creates a shared moment of collective recognition: "we all know who this is." The group is not answering a question — they are acknowledging a truth.

---

## 2. Approved Starter Phrases

Use these starters and rotate through them. No starter should appear more than **2 consecutive times** in a single seed file.

| Starter | Register | Best categories |
|---|---|---|
| "لا تلفّون…" | Playful, conspiratorial | Friends, Funny Chaos |
| "خلونا نكون صريحين…" | Direct, confident | Savage, Deep Exposing |
| "لا حد يمثل…" | Challenging the group | Couples, Embarrassing |
| "ترى فيه واحد…" | Casual, observational | Friends, Majlis GCC |
| "لا تقولون لا…" | Daring, charged | 21+, Savage |
| "بدون لف ودوران…" | Blunt, no-nonsense | Savage, Deep Exposing |
| "خلونا نقولها مثل ما هي…" | Truth-telling, serious | Deep Exposing, Couples |
| "فيه واحد واضح…" | Direct observation | Friends, Embarrassing |
| "لا حد يجامل…" | Confrontational warmth | Couples, Savage |

---

## 3. Forbidden Starter Patterns

The following patterns soften the accusation into speculation and must never be used:

- **"واضح فيه حد"** — too passive, loses the collective pressure
- **"يمكن فيه"** — introduces doubt, kills recognition speed
- **"تحسون فيه"** — deflects the accusation onto feelings rather than facts
- **"منو ممكن"** — returns to question format

---

## 4. Prompt Strength Criteria

A prompt is **strong** only if:

- Players can immediately think of someone specific
- Being chosen creates slight discomfort or a knowing laugh
- It exposes a hidden behavior, reputation gap, or social contradiction
- It feels like: *"we all know who this is"*

A prompt is **weak** if:

- It feels generic (applies equally to everyone)
- It has no consequence if chosen
- It sounds like a casual party question
- It depends on explicit wording to feel bold

---

## 5. Approval Scoring Model

Score each prompt 1–5 on these dimensions. **Approve only if ≥ 4 in at least 3 dimensions.**

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| **Specificity** | Generic — anyone | A subset of people | Immediately names one person |
| **Social risk** | Nobody feels called out | Mild awareness | Someone will shift in their seat |
| **Recognition speed** | > 5 seconds to decide | 2–5 seconds | Instantaneous — first face arrives immediately |
| **Emotional impact** | No reaction | Mild smiles | Laughter, gasps, or a story breaks out |
| **Replay value** | Repetitive the second time | Works again with different groups | Always lands regardless of group |

## 6. Per-Category Tone Definitions

### 6.1 `friends`
| | |
|---|---|
| **Entry tone** | Warm, conspiratorial — the group is sharing an inside joke |
| **Accusation shape** | Observable social habit that belongs to exactly one person |
| **Starter preference** | "لا تلفّون…" / "ترى فيه واحد…" / "فيه واحد واضح…" |
| **Avoid** | Hard starters like "بدون لف ودوران" — too aggressive for this register |

### 6.2 `couples`
| | |
|---|---|
| **Entry tone** | Intimate, knowing — the group has watched this dynamic for years |
| **Accusation shape** | Relationship imbalance, emotional pattern, or control behavior |
| **Starter preference** | "لا حد يمثل…" / "خلونا نقولها مثل ما هي…" / "لا حد يجامل…" |

### 6.3 `embarrassing`
| | |
|---|---|
| **Entry tone** | Delighted cringe — the group is gleefully pointing at a past mistake |
| **Accusation shape** | Specific past or recurring social blunder |
| **Starter preference** | "فيه واحد واضح…" / "لا تلفّون…" / "بدون لف ودوران…" |

### 6.4 `savage`
| | |
|---|---|
| **Entry tone** | Blunt, relentless — no mercy, no softening |
| **Accusation shape** | Hypocrisy or internal contradiction — the behavior contradicts the stated self-image |
| **Starter preference** | "بدون لف ودوران…" / "خلونا نكون صريحين…" / "لا تقولون لا…" |

### 6.5 `deep_exposing`
| | |
|---|---|
| **Entry tone** | Quiet, serious — no theatrics |
| **Accusation shape** | Hidden internal state or suppressed truth — something circling unspoken for months |
| **Starter preference** | "خلونا نقولها مثل ما هي…" / "بدون لف ودوران…" |
| **Avoid** | Funny starters — they kill the gravity |

### 6.6 `majlis_gcc`
| | |
|---|---|
| **Entry tone** | Warm, traditional, culturally affectionate — the accusation is a badge of honor |
| **Accusation shape** | Cultural habit or generational behavior that the group admires or playfully recognizes |
| **Starter preference** | "ترى فيه واحد…" / "لا تلفّون…" only |
| **Avoid** | Aggressive starters — "بدون لف ودوران", "لا تقولون لا" break the cultural register |

### 6.7 `funny_chaos`
| | |
|---|---|
| **Entry tone** | Absurdist, high energy — the behavior is cinematic exaggeration |
| **Accusation shape** | Everyday behavior turned into a movie scene or national incident |
| **Starter preference** | "لا تلفّون…" / "فيه واحد واضح…" |

### 6.8 `age_21_plus`
| | |
|---|---|
| **Entry tone** | Charged, conspiratorial — the group is acknowledging things they've noticed but never said |
| **Accusation shape** | Hidden adult life, reputation contrast, controlled danger |
| **Starter preference** | "لا تقولون لا…" / "لا حد يجامل…" / "بدون لف ودوران…" |
| **Note** | Hints at adult behavior through insinuation, not explicit description |

## 7. Transformation Examples

### Friends
- **Before:** "من يمسك الجوال أكثر وقت في الجلسة؟"
- **After:** "فيه واحد واضح… جواله أهم ضيف في الجلسة وما يكاد يحط يده إلا ليمسكه ثاني مرة"

### Couples
- **Before:** "من يصعد بسرعة أكثر عند الخلاف؟"
- **After:** "لا حد يمثل… فيه واحد لما يجي خلاف حتى لو بسيط درجة حرارته ترتفع قبل ما الثاني يكمل جملته"

### Embarrassing
- **Before:** "من يتكلم على شخص ويلتفت يلاقيه وراه مباشرة؟"
- **After:** "بدون لف ودوران… فيه واحد صار معه موقف يتكلم على أحد ثم يلتفت ويلقاه واقف خلفه بكل هدوء"

### Savage
- **Before:** "من يمدحك أمام الكل ويتكلم عليك لما تغيب؟"
- **After:** "خلونا نكون صريحين… فيه واحد وجهه مبتسم لك وكلامه عنك مختلف لما تطيح"

### Deep Exposing
- **Before:** "من يحمل شيء ثقيل هالفترة ومحد في الجلسة يعرف؟"
- **After:** "خلونا نقولها مثل ما هي… فيه واحد شايل شيء هالفترة وما عرض على أحد وما أحد سأل"

### 21+
- **Before:** "من يغازل ويدعي بعدها إنه ما كان يقصد؟"
- **After:** "لا تقولون لا… فيه واحد يغازل بكل تركيز وبعدها يقول 'أنا بس كنت لطيف' بعين ثابتة"

### Majlis GCC
- **Before:** "من أكثر شخص يعرف يفرق بين أنواع القهوة الخليجية؟"
- **After:** "ترى فيه واحد… ما تقدم له قهوة إلا ويقولك من أي منطقة هي وإيش ناقصها"

### Funny Chaos
- **Before:** "من يبالغ في توديع الناس وكأنه مسافر لقارة ثانية؟"
- **After:** "فيه واحد واضح… توديعه لك من الغرفة المجاورة فيه من الإحساس أكثر من توديع المطار"

## 8. Updated Pre-Commit Checklist

*Supersedes Mission 4 §8. Use this checklist for all new and rewritten prompts.*

- [ ] Does **NOT** end with `؟` — prompts are declarative, never questions
- [ ] Starts with one of the 9 approved starter phrases (§2)
- [ ] Does **NOT** start with a forbidden pattern (§3)
- [ ] Scores ≥ 4 in at least 3 of the 5 strength dimensions (§5)
- [ ] The accused behavior is observable or inferable — not hypothetical
- [ ] `packId` matches a valid `CategoryKeys` constant
- [ ] `intensity` and `ageRating` are within the category matrix (Mission 4 §6)
- [ ] Does not semantically duplicate an existing prompt in the same pack
- [ ] Does not name a real person, place, or brand
- [ ] Arabic is natural Gulf dialect — not formal MSA
- [ ] `textEn` is present or explicitly set to `""` — never omitted
- [ ] `id` follows `q_{packId}_{NNN}` format and has not been used before

---

## 9. Language Style Reference

Use these constructions freely:
- "فيه واحد" / "فيه حد"
- "السالفة"
- "يلعبها"
- "هالجلسة" (not "هذه الجلسة" — too formal)
- "تطيح" (leave), "يتحول" (changes)

Avoid:
- Formal MSA constructions ("يقوم بـ", "يتمتع بـ")
- Heavy dialect-specific slang that doesn't travel across GCC countries
- Sentence endings with `؟` or formal punctuation
